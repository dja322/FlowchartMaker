# Tkinter flowchart

This is a basic flowchart maker that is editable from the CLI.
Meant for basic nodes and arrows pointing between nodes.

Built using Python and tkinter.

## Features
 - Create Node
 - Make connections between nodes
 - edit connections between nodes

## Requirements
I used python 3.12, not tested with older versions

## Installation
Clone repository and run FlowMaker.py
